# 프로젝트 소개

TBD 를 사용하여 여러 프로젝트를 하나의 소스로 관리하는 전략을 소개합니다.

# 프로젝트 설명

# 확인방법

project 별로 branch 가 존재합니다.

| 프로젝트   | 브랜치    | 비고              |
| ---------- | --------- | ----------------- |
| 개발용     | master    | trunk base branch |
| 프로젝트 A | project-a |                   |
| 프로젝트 B | project-b |                   |

# 실행 방법

1. 각 branch 로 이동
2. 개발 서버 실행
   ```bash
   $ yarn
   $ yarn dev
   ```
